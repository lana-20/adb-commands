### ADB Commands:

__Change battery level from CLI__
- _adb shell dumpsys battery set level <battery_percent_value>_
  - eg, _adb shell dumpsys battery set level 20_
  

__Mimic device is charging__
- adb shell dumpsys battery ...
  - eg, ...
